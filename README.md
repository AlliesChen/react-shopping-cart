# React Shopping Cart

This app is built with React, TypeScript, and Tailwind CSS.

Origin: https://youtu.be/lATafp15HWA

*It's Bootstrap rather than Tailwind which the UI framework that is used in The tutorial.

## Use dev server

`npm run dev`

## Build

`npm run build`

## Preview

`npm run preview`
